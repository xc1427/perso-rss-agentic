# perso-rss-agentic

Generate RSS feeds in an agentic way for personal use. Sources are declared in YAML, scrapers are auto-generated by a Claude agent, and the resulting RSS XML is published via GitHub Pages.

## Feeds

Browse the published feeds at the GitHub Pages index:

**https://xc1427.github.io/perso-rss-agentic/**

Each source's XML is served at `https://xc1427.github.io/perso-rss-agentic/{slug}.xml`.

## Adding a Source

Drop a YAML file in `sources/`. The pipeline auto-generates a scraper on the next run and commits it. See [ARCHITECTURE.md](ARCHITECTURE.md) for details.
